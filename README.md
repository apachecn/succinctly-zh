## 下载

### Docker

```
docker pull apachecn0/succinctly-zh
docker run -tid -p <port>:80 apachecn0/succinctly-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install succinctly-zh
succinctly-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g succinctly-zh
succinctly-zh <port>
# 访问 http://localhost:{port} 查看文档
```
